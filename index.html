Resumen de AJAX

¿Qué es AJAX?

AJAX (Asynchronous JavaScript And XML) es una técnica que permite a una página web enviar y recibir datos del servidor sir recargar la página completa. AJAX permite actualizar solo una parte de la página,
de manera rápida y silenciosa. Esto hace que las paginas sean más rápidas y dinámicas. Aunque incluya XML, hoy en día se usa principalmente con JSON.

¿Para qué se usa AJAX?

• Cargar datos sin recargar la página. Ejemplo: ver más comentarios en Facebook.
• Enviar formularios sin refrescar. Ejemplo: registrarte o iniciar sesión sin que la página se recargue.
• Mostrar resultados al escribir. Ejemplo: sugerencias de Google cuando escribes.
• Actualizar partes de un sitio web. Ejemplo: el carrito de compras en Amazon.

¿Cómo funciona AJAX?

1. La página web envía una petición al servidor.
2. El servidor procesa esa petición.
3. El servidor devuelve una respuesta.
4. JavaScript recibe esa respuesta.
5. La página se actualiza automáticamente sin recargarse.

Ejemplo:

            <button onclick="cargarMensaje()">Cargar mensaje</button>
            <p id="resultado"></p>
            

            <script>
                function cargarMensaje() {
                    fetch("mensaje.php")   // Se llama a un archivo en el servidor
                        .then(respuesta => respuesta.text())
                        .then(data => {
                            // Actualiza la página
                            document.getElementById("resultado").innerHTML = data; 
                        });
                }
            </script>
            

            <?php
                echo "Hola desde el servidor!";
            ?>
